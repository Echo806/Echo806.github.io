Linux命令

2026-03-12

whoami
cd (change directory)
sudo(SuperUser Do)
rm(remove) -r(recursive) -f(force)
ls(list) -l(long) -r(recursive) -a(all)
mkdir(make directory)
pwd(print working directory)
touch(触碰一下文件，更新它的时间戳)(可以用来创造文件)
cat(concatenate)(拼接)（后面只接一个文件可以用来读取文件）
mv(move)
cp(copy)
head(看头) -n1（看第一行）
tail(看尾) -n1 (看最后一行)

chown(change owner)很多时候要搭配sudo来使用
chown 用户：组 文件
chown Tom:developers test.txt

chmod(change mod)
chmod 700 test.txt （改文件权限） r(read) w(write) x(execute)
第一个数字之和表示owner权限
第二个数字之和表示group权限
第三个数字之和表示others权限
4: Read permission
2: Write permission
1: Execute permission
0: No permission

chomd -R （revursive）递归文件夹改变所有权限

grep (Global Regular Expression Print)
grep “ERROR” log.txt

dmesg(diagnostic message) 诊断信息（系统/内核输出的消息)
dmesg | grep usb
dmesg | grep error
sudo dmesg | grep -Ei ‘fail|error’ >~/project/boot_issues.txt
(The grep command’s -i option makes the search case-insensitive.To search for multiple patterns at once (like fail OR error), you can use grep -E ‘pattern1|pattern2’.)

一些概念

x11,wayland 是图形系统

Xfce,cinnamon，KDE(指的是KDE Plasma)，Gnome是桌面环境（desktop environment）

KDE（社区 / 项目组织）是长期开发 Linux 桌面软件的社区组织。KDE Plasma 是 KDE 做的桌面环境（你看到的图形桌面界面）。所以很多人说 KDE，其实常指 KDE Plasma。

Hyprland 更准确是：Wayland Compositor（Wayland 合成器）

合成器 = 把各个窗口画面拼接起来，并最终显示到屏幕上的程序。

壁纸
+ 浏览器窗口
+ 微信窗口
+ 鼠标指针
+ 阴影
+ 透明效果
+ 动画
= 最终屏幕画面

把很多图层拼成一个完整画面。

这就叫：

合成（Compose）

Canonical（公司名，Ubuntu 背后的商业公司）

Waydroid，它是 Linux 上运行 Android 应用的一种方案。你可以理解成：

在 Linux 里放一个 Android 系统环境，然后运行安卓 App。

但 Waydroid 通常比传统模拟器更轻。

不是传统模拟器，Waydroid 更像：

把 Android 放进 Linux 容器（轻量隔离环境）里运行。

Waydroid 可以理解成“专门跑安卓的桌面容器方案”。

KVM = Linux 内核里的虚拟机技术。（内核 = 操作系统最核心部分）

它让 Linux 电脑可以变成虚拟机宿主机（运行其他系统）。比如你可以在 Linux 上开：

  • Windows 虚拟机
  • Ubuntu 虚拟机
  • Android x86 虚拟机

你可以理解成：

KVM = Linux 自带的 VMware / VirtualBox 级能力（而且更底层）

Wine 不是虚拟机，也不是模拟器。 它是 兼容层（Compatibility Layer，翻译：把 Windows 接口转换成 Linux 能理解的接口）。

Windows 软件以为自己在 Windows 上运行，实际上 Wine 在中间“翻译”。

Wine名字意思是:Wine Is Not an Emulator

Wine 不是模拟器。

Wine

  • 常见 Windows 软件
  • 想省资源

KVM 虚拟机

适合：

  • 办公软件兼容要求高

Waydroid

适合：

  • 安卓 App

tmux 是一个 终端复用器（让一个终端窗口变成多个终端，并且会话可持续保存的工具）。

Mesa 是 Linux 世界最重要的 开源图形驱动与图形库集合。 它负责让系统和显卡正确工作、运行 3D 图形、游戏、桌面加速等。

而 Vulkan 是一种 图形接口标准（让游戏/软件调用 GPU 的方式）

所以两者关系是：

Vulkan 是规则 / 接口标准，Mesa 是 Linux 上实现这些规则的重要开源方案之一。

类比最容易懂：

Vulkan = 交通法规

规定：

  • 红灯停绿灯行
  • 怎么开车
  • 怎么变道

Mesa = 汽车制造厂

把这套规则落实成真正能开的车。

游戏调用 Vulkan：

游戏 -> Vulkan 命令 -> Mesa 驱动 -> GPU 显卡

Mesa 帮你把 Vulkan 指令翻译成显卡能执行的动作。

VA-API =Video Acceleration API

VA-API 是 Linux 常见的 视频硬件加速接口。 它让播放器、浏览器、录屏软件把视频解码/编码工作交给显卡（GPU）或专用硬件处理，而不是全靠 CPU。

你可以把它理解成：电脑里的“视频专用加速通道”。

Proton 是 Valve 为 Linux 游戏打造的 Windows 游戏兼容层。 它让很多原本只能在 Windows 上运行的游戏，可以直接在 Linux 上玩。

Btrfs（读作 “Butter FS” 或按字母念 B-tree FS）是一种 Linux 文件系统。

目标是替代一些传统文件系统的不足，比如 ext4 太稳但功能偏传统。
