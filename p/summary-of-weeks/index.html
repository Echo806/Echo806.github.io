配置太旧了ubuntu和mint上apt或者flatpak安装的steam启动杀戮尖塔2都会闪退，但是snap安装的steam启动杀戮尖塔2倒是没问题。

报错如下：

ERROR: ld.so: object '/home/runrun/.steam/debian-installation/ubuntu12_32/gameoverlayrenderer.so' from LD_PRELOAD cannot be preloaded (wrong ELF class: ELFCLASS32): ignored.
ERROR: ld.so: object '/home/runrun/.steam/debian-installation/ubuntu12_64/gameoverlayrenderer.so' from LD_PRELOAD cannot be preloaded (wrong ELF class: ELFCLASS64): ignored.

这个意思是 Steam 的游戏叠加层（Overlay）动态库无法加载，原因是 ELF 类别不匹配（32 位 vs 64 位）。

这是 apt 版 Steam 常见问题，因为它默认依赖系统库，而 snap 版自带运行时

还有Vulkan / Intel GPU 警告：

MESA-INTEL: warning: Haswell Vulkan support is incomplete

我用的是 Intel HD Graphics Haswell，Mesa Vulkan 驱动支持不完整。

这通常会导致用 Proton 或 Vulkan API 的游戏闪退。

不折腾了，用下snap安装的steam糊弄吧，等硬件价格降下来装台强强电脑玩。
