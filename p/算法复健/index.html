找到字符串中所有字母异位词

给定两个字符串 s 和 p，找到 s 中所有 p 的 异位词 的子串，返回这些子串的起始索引。不考虑答案输出的顺序。

创建两个哈希映射，key是字母，value是出现次数

第一个哈希映射传入p的字母和出现次数

第二个哈希映射就是用来滑动窗口记录当前字串的字母和出现次数

当向右边延伸的时候，出现某个字母出现次数大于第一个哈希映射里面那个字母出现的次数时，把左边一直删直到删到了和右边这个字母一样的位置。

每次传入向右边传入新字母后，判断一下字串和p串长度是不是一样大，是的话窗口左边的索引就是所求。

最小覆盖子串

创建一个哈希集合cnt（取名为need也很恰当），表示还需要什么字母多少次。

先遍历字符串t，填入cnt，用cnt_variety记录出现了多少种字母

再滑动窗口遍历s，每出现一个字母，就把cnt中key为这个字母的value减一

再判断cnt中key为字母现在对应的value是不是0（如果是不在t中的字母，肯定value是负数，不用考虑啦）

是，就把所需字母cnt_variety减一。

判断字串是否满足覆盖，只要判断cnt_variety是否等于0

那怎么判断是不是最小覆盖？

从最左边开始收缩

while(cnt_variety==0)
{
    if(right-left<ans_right-ans_left)
    {
        ans_left=left;
        ans_right=right;
    }
    if(cnt.get(s[left])==0)
    {
        cnt_variety++;
    }
    cnt.put(s[left],cnt.get(s[left])+1);
    left++;
}

最大子数组和

一个想法，连续数组前面部分的贡献一定要大于0，如果小于0，那不如不要了

用total记录连续数组的和

从左到右遍历数组

int ans=nums[0];
int total=0;
for(int x:nums)
{
    total+=x;
    if(total>ans)
    {
        ans=total;
    }
    if(total<0)
    {
        total=0;
    }
}
